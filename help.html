üîß Troubleshooting & FAQ

I can't log in to my account +

Common solutions:

  • Double-check your Staff ID or Business ID for typos
  • Ensure you're using the correct login tab (Staff/Company/Super Admin)
  • Clear your browser cache and cookies
  • Try using a different browser
  • Contact your manager for account verification
Payment processing isn't working +

Check these items:

  • Verify internet connection is stable
  • Ensure payment terminal is connected and powered on
  • Check if payment gateway settings are configured
  • Try processing a smaller test transaction
  • Contact your payment processor if issues persist
Printer is not working +

Troubleshooting steps:

  • Check printer power and paper levels
  • Verify printer is connected (USB/Network)
  • Test print from printer settings
  • Restart the printer and POS system
  • Check for driver updates
Menu items are missing or incorrect +

Resolution steps:

  • Refresh the browser page
  • Check if items are marked as "available" in settings
  • Verify menu synchronization is working
  • Contact business administrator to update menu
  • Check for any scheduled menu changes
How do I process a refund? +

Refund process:

  • Access the original transaction in reports
  • Click on the refund option
  • Enter refund amount and reason
  • Process through the payment terminal
  • Print refund receipt for customer
  • Note: Manager approval may be required
System is running slowly +

Performance optimization:

  • Close unnecessary browser tabs
  • Clear browser cache and cookies
  • Check internet connection speed
  • Restart the browser or device
  • Contact IT support if issues persist
